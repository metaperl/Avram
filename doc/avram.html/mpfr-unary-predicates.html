D.14.5 mpfr unary predicates

Each of these predicates takes an mpfr number as an argument and performs a logical operation. If the result is true, it returns (nil,nil), and otherwise it returns nil.

  • nan_p
  • inf_p
  • number_p
  • zero_p
  • integer_p
