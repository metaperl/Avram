D.5 gsldif

Numerical differentiation of a real valued function of a single real variable can be done by a library function of the form 

     library('gsldif',method)

where method is one of

  • 'backward'
  • 'central'
  • 'forward'
  • 't_backward'
  • 't_central'
  • 't_forward'
