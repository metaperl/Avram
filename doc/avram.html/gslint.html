D.7 gslint

An interface to a selection of numerical integration routines from the GNU Scientific Library is provided by functions of the form 

     library('gslint',q)

where q can be one of 'qng', 'qng_tol', 'qagx', 'qagx_tol', 'qagp', or 'qagp_tol'.
