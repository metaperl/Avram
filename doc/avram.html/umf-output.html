D.17.2 umf output

If no exception occurs, the solution x to the matrix equation M x = b noted previously will be returned if one exists.

The solution is represented as either a list of real numbers as in math, or a list of complex numbers as in complex. Real numbers are returned by the di_* functions, and complex numbers are returned by the zi_* functions.

If no solution exists due to a singular matrix, an empty list is returned. The lack of a solution isn’t treated as an exceptional condition.
