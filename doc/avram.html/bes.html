D.1 bes

An interface to the Bessel functions as defined in the GNU Scientific Library (gsl) is available to virtual code applications by invoking a function of the form

 
library('bes',f)

where f is a character string identifying the Bessel function family. All functions in this library return a floating point number encoded as in math.
